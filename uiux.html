  • ▶ Mobile Prototypes
    • Fitness App for Bachelor's Thesis
      This fitness app was developed specifically for my bachelor’s thesis. In the process, I recreated five key components commonly found in fitness apps, namely, the start screen, navigation bar, add workout function, workout detail view, and calendar, following a scientific design approach. The main frameworks applied were: Nielsen's 10 Usability Heuristics and the Gestalt Principles. To create the app effectively, I first analyzed five existing fitness apps and evaluated them using insights from relevant literature. I then combined this knowledge with design theory to develop the app. After the design phase, I conducted user tests to gather feedback, which confirmed that a scientifically grounded design approach results in an effective and user-friendly experience.

  • ▶ Applications
    • Spotify Spatial Music Viewer
      In this project, I collaborated in a team to link Spotify data with a spatial layer. Our goal was to visualize the most-streamed albums and songs in each European country. We aimed to create an interactive web map that allows users to click on a country to explore its specific trends. Using the Spotify API, we accessed the most up-to-date data, and with Python, we implemented an automated process for continuous data retrieval. When deployed on a server, the trends would update automatically over time. You can read more about this project on my GitHub account, where I have documented it in great detail.
